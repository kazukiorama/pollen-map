# pollen-map

花粉飛散情報の取得と表示用

## データ提供元

本アプリで表示している花粉飛散数データは、**株式会社ウェザーニューズ**が提供する花粉観測オープンデータ（Pollen Counts API）を利用しています。

- API ドキュメント: <https://wxtech.weathernews.com/products/data/api/operations/opendata/getPollenCounts/>
- 提供元: [WeatherNews OpenData](https://wxtech.weathernews.com/opendata/v1/)

データの著作権は株式会社ウェザーニューズに帰属します。
